HTML Reference Page

HTML

In order to begin an html document, a "<html>" is used as the opening and closing tags, which signify the beginning and end of a html code.

Example:

<html>

<p>Example code.</p >

<html>

Final Result:

Example Code.

HTML Tag

Header

For a web page, the "<header>" tag is used for containing all the head elements, including the title, scripts, styles, and more. The "<h1>" tag would be the largest header and the "<h6>" tag would be the smallest.

Example:

<html>

<h1>Heading 1</h1>

<h2>Heading 2</h2 >

<h3>Heading 3</h3 >

<h4>Heading 4</h4>

<h5>Heading 5</h5>

<h6>Heading 6</h6>

<html>

Final Result:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6
Header Tag

Title

In order to have structure in an html document, a "<Title>" tag is needed to declare the main title of the page, to give a viewer an idea of what it is they are looking at.

Example:

<html>

<head>

<title> Example! </title>

</head>

<html>

Final Result:

HTML Reference Example! Title Tag

Body

In an html document, a "<body>" tag is needed to define the main content of the visible html document. It is also referred to as the "<body>" element.

Example:

<html>

<body> Example code! </body>

<html>

Final Result:

Example code! Body Tag

Paragraph

In an html document, a "<p>" tag is needed to define the main paragraph element of the visible html document. It is used to write a paragraph using text code.

Example:

<html>

<p> This is an Example code! </p>

<html>

Final Result:

This is an Example code!

Paragraph Tag

Breaks

In an html document, a "<br>" tag is needed to create a break in the code, and is very similar to the enter key on a keyboard. It creates a space between two sentences, and is useful to differentiate between different code lines.

Example:

<html>

<p> This is text contains <br> a line break. </p>

<html>

Final Result:

This is text contains
a line break.

Breaks Tag

Lines

In an html document, a "<hr>" tag defines a themaatic break in an HTML web page. The "<hr>" element can be used to divide content in a HTML page.

Example:

<html>

<p> This has a

<hr>

..line break! </p>

<html>

Final Result:

This has a

line break.

Lines Tag

Comments

In an html document, a "<!--...-->" tag is used to insert comments in the source code. Comments are not displayed in the final web page. Also, comments can be used to explain code.

Example:

<html>

<p> Example code! </p>

<!-- This is a comment. -->

<html>

Final Result:

Example code!

Comment Tag

Images

In an html document, a "<img>" tag defines an image in an HTML web page. It helps define the size and the name of the image in the web page.

Example:

<html>

<img src="smiley.gif" alt="Smiley face" height="42" width="42">

<html>

Final Result:



Images Tag

Lists

In an html document, a "<ul>" tag defines an unordered bulleted list. The tag can be used to order any type of list.

Example:

<html>

<ul>

<li>Coffee</li>

<li>Tea</li>

<li>Milk</li>

</ul>

<html>

Final Result:

  • Coffee
  • Tea
  • Milk


Lists Tag

Tables

In an html document, a "<table >" tag defines an HTML table. An HTML table consists of the "<table >" tag, along with other kinds of elements.

Example:

<html>

<table>

<tr>

<th>Month</th>

<th>Savings</th>

</tr>

<tr>

<td>January</td>

<td>$100</td>

</tr>

</table>

<html>

Final Result:

Month Savings
January $100


Tables Tag

Blocks

In an html document, a "<div>" tag defines a section or a division in an HTML document.The "<div>" tag can also be used to group block-elements to form them in CSS (styles).

Example:

<html>

<div style="color:#0000FF">

<h3>This is a heading</h3>

<p>This is a paragraph.</p>

</div>

<html>

Final Result:

This is a heading

This is a paragraph.



Blocks Tag
