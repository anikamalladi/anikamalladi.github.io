HTML

In order to begin an html document, a "<html>" is used as the opening and closing tags, which signify the beginning and end of a html code.

Example:

<html>

<p>Example code.</p >

<html>

Final Result:

Example Code.

Header

For a web page, the "<header>" tag is used for containing all the head elements, including the title, scripts, styles, and more. The "<h1>" tag would be the largest header and the "<h6>" tag would be the smallest.

Example:

<html>

<h1>Heading 1</h1>

<h2>Heading 2</h2 >

<h3>Heading 3</h3 >

<h4>Heading 4</h4>

<h5>Heading 5</h5>

<h6>Heading 6</h6>

<html>

Final Result:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Title

In order to have structure in an html document, a "<Title>" tag is needed to declare the main title of the page, to give a viewer an idea of what it is they are looking at.

Example:

<html>

<head>

<title> Example! </title>

</head>

<html>

Final Result:

HTML Reference Example!

Body

In an html document, a "<body>" tag is needed to define the main content of the visible html document. It is also referred to as the "<body>" element.

Example:

<html>

<body> Example code! </body>

<html>

Final Result:

Example code!

Paragraph

In an html document, a "<p>" tag is needed to define the main paragraph element of the visible html document. It is used to write a paragraph using text code.

Example:

<html>

<p> This is an Example code! </p>

<html>

Final Result:

This is an Example code!

Breaks

In an html document, a "<br>" tag is needed to create a break in the code, and is very similar to the enter key on a keyboard. It creates a space between two sentences, and is useful to differentiate between different code lines.

Example:

<html>

<p> This is text contains <br> a line break. </p>

<html>

Final Result:

This is text contains
a line break.

Lines

In an html document, a "<hr>" tag is defined as a line that can seperate two lines of code. It is very useful for seperating different paragraphs of code in an HTML document.

Example:

<html>

<p> This is text contains <hr> a line break. </p>

<html>

Final Result:

This is text contains

a line break.
